# Fish shell

* `apt install fonts-powerline`
* Install fish on the shell and setup **fisher**
* `curl -Lo ~/.config/fish/functions/fisher.fish --create-dirs git.io/fisher`
* `fisher edc/bass omf/theme-bobthefish`
* `chsh -s /usr/bin/fish `
* `~/.config/fish/config.fish`
